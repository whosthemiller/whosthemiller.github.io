# Photo index for mouse trail

Put your work images in the **`images/`** folder (same level as `index.html`).

**Match the filenames to the list below**, or edit this file:

- To use **your filenames**, replace the `photos` array so each entry’s `src` is the path to your file, e.g.  
  `"images/your-exact-filename.jpg"`
- The index above uses `images/1.jpg`, `images/2.jpg`, … `images/10.jpg`. If your files are named that way, the trail will show them. Add more entries if you have more images.

Only images that load successfully are used; wrong paths are skipped (no broken icons).
